Frequently Asked Questions

Top and most frequently asked questions about the course.

What's the diffrence between a flexbox and a grid?

Flexbox is a layout system that allows you to display elements in a row or column. Grid is a layout system that allows you to display elements in a two dimensional grid.

What's the diffrence between a bootstrap and a tailwind?

Both of them are frameworks that help you to build a website faster. But, they follow different ways to solve the puzzle. Bootstrap follows premade components based design, so we don't need to design the UI from scratch. While Tailwind follows utility classes, so we need to design the UI from scratch. With Tailwind, we have more granular control over the styling of our components.

What is CSS Box Model?

In CSS, the term "box model" is used when talking about design and layout. The CSS box model is essentially a box that wraps around every HTML element. It consists of: margins, borders, padding, and the actual content.

What is Semantic tags in CSS?

Semantic HTML elements are those that clearly describe their meaning in a human- and machine-readable way. Elements such as code header , footer and article are all considered semantic because they accurately describe the purpose of the element and the type of content that is inside them
